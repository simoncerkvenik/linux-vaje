# 08 - Network Basics

## Purpose
This file contains basic notes about networking concepts in Linux and IT.

## Important concepts

### IP address
An IP address identifies a device on a network.

Example:
192.168.1.10

### Subnet mask
A subnet mask defines which part of the IP address belongs to the network and which part belongs to the host.

Example:
255.255.255.0

### Gateway
A gateway is the device that connects the local network to other networks, such as the internet.

Example:
192.168.1.1

### DNS
DNS translates domain names into IP addresses.

Example:
google.com -> 142.250.x.x

### LAN
A local area network is the internal network in a home or office.

### WAN
A wide area network connects networks over larger distances, usually through an internet provider.

### `ping`
Tests connectivity between devices.

Example:
ping 8.8.8.8
