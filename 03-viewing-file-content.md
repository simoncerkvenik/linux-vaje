# 03 - Viewing File Content

## Purpose
This file contains basic notes about commands used to view file content in Linux.

## Common commands

### `cat`
Shows the full content of a file.

Example:
cat file.txt

### `less`
Opens a file in scrollable view.

Example:
less file.txt

### `head`
Shows the first lines of a file.

Example:
head file.txt

### `tail`
Shows the last lines of a file.

Example:
tail file.txt

### `wc -l`
Counts the number of lines in a file.

Example:
wc -l file.txt
