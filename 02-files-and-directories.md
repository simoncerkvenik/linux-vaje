# 02 - Files and Directories

## Purpose
This file contains notes and examples about working with files and directories in Linux.

## Common commands

### `mkdir`
Creates a new directory.

Example:
    mkdir test_folder
### `touch`
Creates an empty file.

Example:
touch test.txt

### `cp`
Copies a file.

Example:
cp test.txt copy.txt

### `mv`
Moves or renames a file.

Example:
mv test.txt new_name.txt

### `rm`
Removes a file.

Example:
rm new_name.txt

### `rmdir`
Removes an empty directory.

Example:
rmdir test_folder
