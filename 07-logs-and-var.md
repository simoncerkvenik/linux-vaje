# 07 - Logs and /var

## Purpose
This file contains basic notes about the `/var` directory and common log files in Linux.

## Important paths

### `/var`
Stores variable data such as logs, cache, and spool files.

Example:
/var

### `/var/log`
Contains system log files.

Example:
/var/log

### `auth.log`
Contains authentication and sudo-related logs.

Example:
/var/log/auth.log

### `syslog`
Contains general system messages.

Example:
/var/log/syslog
