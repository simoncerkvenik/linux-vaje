# 05 - Aliases and Shortcuts

## Purpose
This file contains basic notes about useful Linux shortcuts and command helpers.

## Common commands and shortcuts

### `alias`
Creates a shortcut for a command.

Example:
alias ll='ls -lah'

### `history`
Shows previously used commands.

Example:
history

### `clear`
Clears the terminal screen.

Example:
clear

### `Ctrl + L`
Clears the terminal screen quickly.

Example:
Press Ctrl + L

### `!!`
Repeats the last command.

Example:
sudo !!
